Das Matrizes Matemáticas

🎯 Bem-vindo ao Desafio das Matrizes!

Você enfrentará 5 fases épicas, cada uma com bombas distribuídas segundo uma lei de formação matemática única.

Use sua lógica e conhecimento matemático para identificar padrões, evitar as bombas e conquistar todos os campos seguros!

⚡ Cada fase é um novo desafio lógico!

🎮 Como Jogar

  • Clique nos campos para revelar espaços seguros
  • Se clicar em uma bomba... 💥 Game Over!
  • Descubra todos os campos livres para vencer a fase
  • Cada fase segue uma lei matemática diferente

🧮 As Fases

  • Fase 1: Diagonais - bombas nas diagonais principal e secundária
  • Fase 2: Linhas pares - bombas em todas as linhas pares
  • Fase 3: Matriz triangular - bombas na parte superior
  • Fase 4: Função quadrática - bombas onde i² - 3j < 0
  • Fase 5: Padrão complexo - múltiplas condições matemáticas

💡 Dicas Estratégicas

  • Observe os padrões matemáticos de cada fase
  • Use o conhecimento de matrizes e coordenadas
  • Pense logicamente antes de cada clique
  • As bombas nunca são aleatórias - sempre seguem a lei da fase!
